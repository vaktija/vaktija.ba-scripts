# PDF

## Required tools:

### wkhtmltopdf
https://wkhtmltopdf.org/downloads.html

#### ghostscript
https://www.ghostscript.com/download.html

or

#### pdftk

```
sudo apt install pdftk
```

## Run local web server:

```
cd server
yarn install
yarn start
```

## Run script to generate pdf files (script year):

```
./generate-pdf.sh 2022
```
